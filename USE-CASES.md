# USE-CASES.md

The bananas list. Real things people in your shoes have built using exactly this stack.

---

After Sunday you have the operating system. Here is what people in your shoes have built using exactly this stack. None of these require new tools, new skills, or new subscriptions beyond what you set up Sunday. They are all reachable from where you will be at 5pm ET.

The list is intentionally long. Skim it, dog-ear the ones that make you stop and think "wait, I want that," and then build one. The italicized use cases are confirmed running today on Tyler's daily fleet. The rest are the obvious adjacent unlocks people have built with this same stack on YouTube, in workshops, and in our community.

Layer references map to [the four-layer journey](sara-curriculum/THE-JOURNEY.md): L1 = Infrastructure, L2 = Senses, L3 = Triggers, L4 = Autonomy.
