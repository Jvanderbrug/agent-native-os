# Sara's Workshop Curriculum (Live-Tested April 2026)

This is a parallel curriculum to the main `guides/` directory. Sara walked through every one of these 14 components herself, as a student, and documented every failure mode she hit.

The architecture is 4 layers + a Capstone:

- **Layer 1 (Components 0-3) — Infrastructure:** where the agent lives, thinks, and stores secrets
- **Layer 2 (Components 4-6) — Senses:** how the agent sees the world (Exa, Firecrawl, Gmail/Calendar/Drive)
- **Layer 3 (Components 7-8) — Triggers:** turning capability into one-word commands
- **Layer 4 (Components 9-12) — Autonomy:** the agent runs without you (scheduling, delivery, remote control, progressive autonomy)
- **Capstone (Component 13 + `/build`):** a personal morning brief system with sources, memory, delivery, and a full write-up surface

See [THE-JOURNEY.md](THE-JOURNEY.md) for the full 4-layer story.

## How to use this curriculum

This is the path Sara recommends. The main `guides/` directory has a different structure (numbered 00-10) and works fine too. Pick whichever resonates — your Claude can walk you through either.
