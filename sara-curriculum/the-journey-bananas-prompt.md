## The Prompt

```
For students entering the AI Build Lab Claude Code Workshop — a one-day intensive
that teaches non-technical professionals to install an Agent-Native Operating System
on their own machine — create a "Workshop Journey" map. This infographic appears at
the very beginning of the student guide as a mental scaffold so students understand
where they are, where they're going, and why each of the 13 components exists in
the order it does. It needs to feel both architecturally serious and warmly inviting.

Create a vertical conceptual architecture diagram in the form of a stacked-layer
infographic. The aesthetic is dashboard-meets-blueprint: the visual language of a
clean system architecture diagram softened by educational warmth. Tall portrait
poster format suitable for inclusion in a student handbook, slide deck, and Notion
course page.

The diagram visualizes four stacked layer cards plus one floating capstone card,
read deliberately bottom-up. Render the following text verbatim. At the very top
of the canvas, in a header band, place the title "The Workshop Journey" with the
subtitle "Four layers stacked on top of each other. Each one unlocks the next."
underneath. Below that, suspended above the four-layer stack with a clear gap and
a soft glow, place the capstone card containing the small tag "CAPSTONE", the
heading "Agents that build agents", and the single line "13 · The /build skill".
Then, descending the canvas, place four equally-sized layer cards in this order
from top to bottom: Layer 4 card with tag "LAYER 4", name "Autonomy", description
"How the agent runs without you.", and components list "9 · Scheduling", "10 ·
Delivery channels", "11 · Remote control", "12 · Progressive autonomy". Layer 3
card with tag "LAYER 3", name "Triggers", description "How you activate the
agent.", and components list "7 · Custom slash commands", "8 · Daily briefing
config". Layer 2 card with tag "LAYER 2", name "Senses", description "How the
agent sees the world.", and components list "4 · Exa MCP → the public web",
"5 · Firecrawl → full web pages", "6 · Gmail · Calendar · Drive → your life".
Layer 1 card at the foundation with tag "LAYER 1", name "Infrastructure",
description "Where the agent lives and thinks.", and components list "0 ·
Working with Claude Code", "1 · CLAUDE.md → standing instructions", "2 · Obsidian
Vault → workspace", "3 · 1Password → secrets". A footer band at the very bottom
contains the italic line "Read it bottom-up. Each layer needs the one below it."
above the small attribution "AI Build Lab · Claude Code Workshop".

Compose in portrait orientation at 2048 × 2560 pixels with 96-pixel side margins
and 80-pixel top and bottom margins. Each layer card spans the full content
width, has 32-pixel internal padding, 24-pixel rounded corners, and is separated
from neighboring cards by a 56-pixel vertical gap containing a small upward-
pointing chevron arrow colored to match the destination layer above. Within each
layer card, split horizontally into two zones with a delicate vertical divider
line at 20% opacity: the left zone occupies 40% of card width and stacks the
layer tag, layer name, and italic description; the right zone occupies 60% of
card width, leads with a small "COMPONENTS" label, and lists the components
underneath. The capstone card is narrower at 60% of the content width, centered,
and elevated above Layer 4 by a 64-pixel gap with a soft radial glow behind it
to convey ascendance.

Guide the eye bottom-up — the deliberate inversion of normal reading order is
core to the meaning. Reinforce upward flow with small upward chevron arrows
between cards, each tinted with the accent color of the layer above it. Each
layer card carries a subtle internal gradient that is slightly darker at the
bottom and lighter at the top, creating a visual sense of lift. Layer 1 sits
heaviest and most grounded; the capstone literally floats with a soft drop
shadow beneath it suggesting elevation off the page. The colors warm and
brighten as the eye climbs the stack.

Set the entire composition against a deep midnight gradient background that
fades from charcoal navy #1a1a2e at the top to deep navy #16213e at the
bottom. Overlay a subtle blueprint grid pattern at 6% opacity with 48-pixel
spacing across the canvas, suggesting architectural schematics. Add faint
geometric circuit traces in the corners at 4% opacity to hint at the technical
AI nature without competing with content. A very soft electric-blue radial
glow emanates from behind the capstone card. Apply a fine paper grain texture
at 2% opacity across the entire canvas for warmth.

Typography hierarchy: title "The Workshop Journey" in Montserrat ExtraBold 56pt
white #FFFFFF; subtitle in Open Sans Regular 20pt light gray #E0E0E0; layer
tags "LAYER 1" through "LAYER 4" and "CAPSTONE" in Montserrat SemiBold 14pt
with 2.5pt letter-spacing at 80% white opacity; layer names ("Infrastructure",
"Senses", "Triggers", "Autonomy", "Agents that build agents") in Montserrat
Bold 38pt white #FFFFFF; layer descriptions in Open Sans Italic 18pt at 90%
white opacity; the small "COMPONENTS" labels in Montserrat SemiBold 12pt with
2pt letter-spacing at 70% white opacity; component lines in Open Sans Regular
16pt white #FFFFFF; footer line in Open Sans Italic 16pt light gray #E0E0E0;
attribution in Open Sans Regular 12pt medium gray #9E9E9E. Color palette per
layer using AI Build Lab brand: Layer 1 Infrastructure card in deep slate
navy #1F2937 for foundational grounding; Layer 2 Senses card in brand teal
#4ECDC4 for perception; Layer 3 Triggers card in brand coral #FF6B6B for
action; Layer 4 Autonomy card in brand electric yellow #FFE66D for energy
(use rich saturated yellow with white text maintaining WCAG AA contrast via
slightly darkened yellow base #E6CC4A if needed); capstone card in primary
brand electric blue #00D4FF with a subtle outer glow. All cards have a matte
digital surface, fine paper grain at 3% opacity, and a soft drop shadow at
12-pixel blur, 4-pixel vertical offset, 18% black opacity. The capstone has
a stronger drop shadow at 24-pixel blur with 8-pixel offset and 25% opacity
plus an outer glow at 32-pixel blur in electric blue at 20% opacity. All text
must maintain WCAG AA contrast ratios. Render with flat lighting and crisp
anti-aliased edges on rounded corners and chevron arrows. Output at 4K
portrait resolution 2048 × 2560 pixels in sRGB color space, suitable for
embedding in slide decks, the Notion course handbook, the workshop landing
page, and printed handouts.

[SEED IMAGE INSTRUCTION: Attach the AI Build Lab logo as a reference / seed
image in Google AI Studio before generating. Place the logo subtly at the
bottom-center of the footer band, below the attribution line, sized at
roughly 64 pixels tall.]
```

## How to Generate

1. Open [Google AI Studio](https://aistudio.google.com/) → select Imagen 3 / Nano Banana Pro
2. **Attach the AI Build Lab logo as a seed image** before pasting the prompt
3. Set output: **2048 × 2560 portrait, 4K, sRGB**
4. Paste the prompt above (everything inside the code block)
5. Generate. Re-roll if needed.

## Re-Roll Checklist

Check the first generation for these specific issues — re-roll if any fail:

- [ ] **All 13 component lines render correctly** (Nano Banana Pro is ~97% accurate on text but check every line)
- [ ] **Layer tags read "LAYER 1", "LAYER 2", "LAYER 3", "LAYER 4", "CAPSTONE"** (not numbered weirdly)
- [ ] **Layer names spelled correctly:** Infrastructure, Senses, Triggers, Autonomy
- [ ] **Bottom-up order is correct** — Infrastructure at bottom, Capstone floating at top
- [ ] **Color mapping per layer** matches the spec (navy / teal / coral / yellow / electric blue)
- [ ] **Capstone is visibly elevated** above Layer 4 (gap + shadow + glow)
- [ ] **Upward chevron arrows present** between each layer
- [ ] **AI Build Lab logo visible** in the footer band
- [ ] **Yellow layer text readable** (Layer 4 — yellow + white can blow out; if so, regenerate or accept the darkened yellow base)
- [ ] **No spurious decorative elements** that distract from the architecture (e.g., random icons, faces, mascots)
