# Start Here

Welcome. This repo is your school. The workshop on Sunday installed the teacher (your Claude). Everything you need to learn is in this folder, and your own Claude can walk you through it at your own pace.

## Right now (one-time, ~10 min)

Open Claude Code in this folder (`cd` into the repo, then run `claude`) and paste:

> I just finished the AI Build Lab workshop. Run the `/personalize` slash command from `configs/commands/personalize.md`. Walk me through the get-to-know-you conversation and personalize my CLAUDE.md based on my answers. Take your time — don't let me short-circuit by saying "treat me as a beginner." Force the full conversation.

That's the unlock. Once your CLAUDE.md is personalized, every future session reads from it automatically.

## Then any time after that

Pick a guide from `guides/`. Open Claude Code in this folder. Say:
> Walk me through `guides/04-mcp-servers.md` like I'm new to this. Help me set up the things this guide teaches on my actual machine. Be patient with me; this is my first time.

Your Claude knows what to do.

## What's in this repo

- `configs/` — Personalization questionnaire, starter MCP settings, and the safe / auto / teams mode toggles for your Claude.
- `guides/` — Numbered walk-throughs (00 through 10) covering everything from terminal basics to running an agent OS.
- `blueprints/` — Pre-built reference projects you can clone and adapt.
- `tracks/` — Suggested learning paths for different comfort levels.
- `setup/` — Mac and Windows machine prep — install lists, prereqs, and `verify.sh` checks.
- `templates/` — Drop-in starter files, including the Obsidian/Cairns second-brain starter vault.
- `resources/` — Glossary, keyboard shortcuts, troubleshooting, and Mac/Windows parity reference.
